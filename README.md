# 🎮 Stick Hero Game

A desktop implementation of the classic **Stick Hero** game developed using **Java**, **JavaFX**, and **IntelliJ IDEA**.

## 🚀 Features

* Smooth JavaFX GUI
* Stick extension and bridge mechanics
* Character movement animation
* Score tracking
* Restart functionality

## 🛠️ Technologies

* Java
* JavaFX
* IntelliJ IDEA
* Maven

## ▶️ How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Install Maven dependencies.
4. Run the JavaFX application.
